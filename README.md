# Movie-Recommend-System
Built a Machine Learning web app using Python and Streamlit. The project includes data preprocessing, model training, and saving the model as a .pkl file. An interactive UI allows users to enter inputs and get real-time predictions. The project is organized and version-controlled on GitHub.
